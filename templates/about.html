Motivation of the Project

As the COVID-19 is not contained in the US, the interest in recipes has been increased whereas the interest in restaurants has been decreased. We can see the daily numbers of confirmed cases and deaths in the US, and Google search query trends in each figure below. This recipe recommendation app uses about 57,000 recipes from "allrecipes.com". It is an ingredient-based personalized recommendation system which takes user's ingredient list, preferred ingredients, and dietary restriction. The main purpose of this recommendation is to recommend recipes which use more of the ingredients currently possessed and use less extra ingredients to buy.
